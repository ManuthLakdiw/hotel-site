# 🏨 Hotel Reservation Website - Frontend Interface

A responsive and mobile-friendly frontend interface designed for a modern hotel booking system. This project showcases a clean UI/UX design utilizing the Bootstrap grid system and custom CSS styling to ensure a seamless experience across all devices.

## ✨ Key Features

* **Fully Responsive Design:** Adapts perfectly to mobile, tablet, and desktop screens.
* **Modern UI/UX:** Clean and intuitive interface for an effortless hotel booking experience.
* **Grid Layouts:** Structured and organized content placement using Bootstrap's powerful flexbox grid.
* **Interactive Elements:** Smooth navigation and dynamic components powered by vanilla JavaScript.

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **Bootstrap**
* **JavaScript**

## 💻 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/ManuthLakdiw/hotel-site.git
   ```
2. Open `index.html` in your browser — no build tools needed!
